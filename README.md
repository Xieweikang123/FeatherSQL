# FeatherSQL

基于 Tauri 2 的 SQL 数据库管理工具

## 功能特性

### 数据库支持
- 🔌 **多数据库支持** - 支持 SQLite、MySQL、PostgreSQL、MSSQL (SQL Server)
- 🔗 **连接管理** - 创建、编辑、删除、测试数据库连接，配置信息持久化保存
- 📁 **数据库浏览** - 自动列出数据库和表，支持列表和网格视图，支持表搜索

### SQL 编辑器
- 📝 **Monaco Editor** - 基于 Monaco Editor，支持语法高亮和代码补全
- 🎯 **智能补全** - 自动补全表名、列名，根据数据库类型提供相应语法支持
- ⌨️ **快捷键执行** - 支持 Ctrl+Enter 快速执行 SQL
- 🔄 **SQL 历史** - 自动保存 SQL 执行历史，支持重新加载到编辑器，支持删除历史记录

### 数据查看与编辑
- 📊 **结果展示** - 查询结果表格展示，错误信息提示
- ✏️ **数据编辑** - 支持直接编辑表格数据，撤销/重做功能，保存修改到数据库
- 🎯 **单元格选择** - 支持多单元格矩形选择，方便批量操作
- 📄 **分页显示** - 查询结果分页显示，可调整每页大小（默认 50 行）
- 🔍 **列过滤** - 支持按列过滤和搜索数据
- 📈 **多列排序** - 支持多列排序功能
- 🎨 **表结构查看** - 查看表的详细结构信息（字段名、类型、主键、自增、可空、默认值等）

### 数据导入导出
- 📥 **数据导入** - 支持从 CSV、JSON、Excel 文件导入数据到表
- 📤 **数据导出** - 支持导出查询结果为 CSV、JSON、Excel 格式
- ✅ **选择性导出** - 支持导出全部数据或仅导出选中行

### 界面与交互
- 📋 **日志面板** - 操作日志记录和查看，支持清空日志
- 🎚️ **可调整布局** - 可拖拽调整 SQL 编辑器和结果表格的高度比例，比例自动保存
- 🖱️ **上下文菜单** - 右键菜单提供快捷操作
- 💾 **工作区状态** - 自动保存 SQL 和工作区状态

## 技术栈

- **框架**: Tauri 2.x
- **前端**: React + TypeScript + Tailwind CSS
- **后端**: Rust

## 开发

### 前置要求

- Node.js (v18+)
- Rust (最新稳定版)

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev:tauri
```

### 构建

```bash
npm run build:tauri
```
