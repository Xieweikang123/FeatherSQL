# 图标文件说明

Tauri 需要图标文件才能构建应用。请执行以下步骤：

## 快速解决方案（开发模式）

对于开发模式，你可以创建一个简单的占位图标：

1. 创建一个 512x512 的 PNG 图标文件，命名为 `app-icon.png`，放在 `src-tauri` 目录下
2. 运行：`npx tauri icon` 来生成所有需要的图标格式

## 或者使用在线工具

1. 访问 https://www.favicon-generator.org/ 或类似工具
2. 上传你的图标
3. 下载生成的 `icon.ico` 文件
4. 将其放在 `src-tauri/icons/` 目录下

## 临时解决方案

如果只是想快速测试，可以暂时使用一个简单的单色图标。
